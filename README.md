### Analysis of Algorithms(Background):
    It is helpfull for analysis that which algorithm is best in terms of 
    space and time complexity.

    Example:    Poblem: Find sum of first N natural Number.

    input:  n=3
    output: b   //1+2+3

    input:  n=5
    output: 15  //1+2+3+4+5

    Three different solution are:

        int fun1(int n){
            return n+(n+1)/2;
        }
        -------------
        int fun2(int n){
            int sum =0;
            for(int i=1;i<=n;i++){
                sum+=i;
            }
            return sum;
        }
        ------------
        int fun3(int n){
            int sum =0;
            for(int i=1;i<=n;i++){
                for(int j=1;j<=i;j++){
                    sum++;
                }
            }
            return sum;
        }
